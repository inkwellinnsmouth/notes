# Inkwell Innsmouth - Location Lore

This document catalogs the key locations in the Inkwell Innsmouth universe. All locations must adhere to the core aesthetic (monotone, warped rubber-hose perspective, high-contrast charcoal black & white).

Detailed lore, specific visual designs, and associated images for each location are now housed in their dedicated directories under `locations/`.

## Active Locations

* [The Soggy Square (Town Center)](locations/soggy_square/prompt.md)
* [The "Gilman House" Hotel](locations/gilman_house/prompt.md)
* [Devil Reef Wharves](locations/devil_reef_wharves/prompt.md)
* [The Path Into Innsmouth](locations/path_into_innsmouth/prompt.md)

## Adding New Locations

To add a new location, utilize the `_template` provided in `locations/_template/` to scaffold the necessary files (`prompt.md`, `images/`, `props/`).
